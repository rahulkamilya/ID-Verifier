Our Mission

This is a web application built using React and Vite that allows users to check the validity of their Aadhaar numbers.

Aadhaar is a 12-digit unique identification number issued by the Indian government, and this website provides a quick and easy way to verify whether a given Aadhaar number is valid or not.

Use Cases

->An individual tries some fraudlent activities

->Cases of duplicate adhar number

->Fetching details using a "valid" adhar number

->Identification of indian citizens
